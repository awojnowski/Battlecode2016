Installation Instructions
-------------------------

1. Open `Terminal`
2. `cd` to the `battlecode-scaffold-master` directory
3. Type `rm -rf src`
4. Type `git clone git@github.com:awojnowski/Battlecode2016.git`
5. Type `mv Battlecode2016 src`
6. Run Battlecode!
